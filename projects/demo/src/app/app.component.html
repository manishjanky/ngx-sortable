ngx-sortable

An angular 4 and above component for sorting list supporting drag and drop sort.
Github Report a bug
  • Documentation
  • Demo

1.Installation

        //using npm
         npm install ngx-sortable

        //using yarn
         yarn add ngx-sortable
         yarn install

2.Using with webpack and tsc builds/ angular-cli builds

  • import NgxSortableModule from ngx-sortable
  • add NgxSortableModule to the imports of your NgModule
        
          import {{'{'}} NgxSortableModule {{'}'}} from 'ngx-sortable'
        
          @NgModule({{'{'}}
            imports: [
              ...,
          
              NgxSortableModule
            ],
            ...
          })
          class YourModule {{'{'}} ... {{'}'}}
  • use <ngx-sortable></ngx-sortable> in your templates to add the sortable list in your view like below
  • a tag in you html will look like below with all the config
          <ngx-sortable [items]="items" [name]="'List'" (listSorted)="listSorted($event)">
              <ng-template let-item>
                  <div class="sortable-list-item">
                  {{'{'}}{{'{'}}item{{'}'}}{{'}'}}
                  </div>
              </ng-template>
          </ngx-sortable>
Where content inside <ng-template></ng-template> is the template that will be used for displaying list items i.e the template of the list item. Also the class can be named accordingly this is just an example. Create a class and add it to your root style.css

3.Config

Input
  • items: any[] - array of items that are part of the list.
  • name: string - name of the list.
  • listStyle: any - list styles such as height, width. 
          listStyle = {{'{'}}
            width:'300px', //width of the list defaults to 300,
            height: '250px', //height of the list defaults to 250,
            dropZoneHeight: '50px' // height of the dropzone indicator defaults to 50
            {{'}'}}
Output
  • listSorted($event): Event - when list is sorted emits listSorted event with updated order
    Where $event in the sorted list.
